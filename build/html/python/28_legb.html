28. 命名空间和作用域

命名空间是名称到对象的映射，作用域是命名空间可直接访问的 Python 程序的文本区域（Textual Region）。 “可直接访问” 的意思是：对名称的非限定引用会在命名空间中查找名称。

28.1. LEGB

Python 有四类作用域：

  • 局部（Local）作用域：函数内

  • 封闭（Enclosing）作用域：嵌套函数中的外函数

  • 全局（Global）作用域：函数定义所在模块

  • 内置（Built-in）作用域：Python 内置模块

在作用域中按名称去寻找对象时（Python 中一切皆对象），会按照 LEGB 规则去查找，作用域查找顺序为:

Local -> Enclosing -> Global -> Built-in

如果发生重名，那么谁先被找到就用谁。

1len = len([])       # Global
2def a():
3    len = 1         # Enclosing
4    def b():
5        len = 2     # Local
6        print(len)
7    b()
8a()

28.2. global 和 nonlocal 关键字

如果要 写 Global/Enclosing 变量，需要使用 global / nonlocal 关键字，否则只能 读 。

 1def scope_test():
 2    def do_local():
 3        spam = "local spam"
 4
 5    def do_nonlocal():
 6        nonlocal spam
 7        spam = "nonlocal spam"
 8
 9    def do_global():
10        global spam
11        spam = "global spam"
12
13    spam = "test spam"
14    do_local()
15    print("After local assignment:", spam)
16    do_nonlocal()
17    print("After nonlocal assignment:", spam)
18    do_global()
19    print("After global assignment:", spam)
20
21scope_test()
22print("In global scope:", spam)

After local assignment: test spam
After nonlocal assignment: nonlocal spam
After global assignment: nonlocal spam
In global scope: global spam
